Project Overview 

We want to create a clickable mobile app prototype in Figma for a concept dating platform called: 

SayYes Connect 

This is NOT a production build. 

This is a high-quality visual and interaction prototype only. 

Goal: 

Feels like real app 

Clean premium UI 

Fully clickable flow 

Used for internal presentation/demo.

Platform Style 

Mobile-first prototype. 

Use: 

iPhone 14 frame (or equivalent modern mobile frame) 

Design must feel: 

modern 

premium 

intentional 

minimal 

calm and refined. 

Avoid: 

playful dating visuals 

hearts or cliché romance icons 

bright flashy colors.

Important UX principle 

The profile should not immediately show the user’s photo. 

Instead, the introduction should prioritise values and intentions first. 

Initial View (Step 1) 

Display a profile card showing: 

Name & Age 

Example: 
Rachel, 34 

Sections visible immediately 

Values Alignment 
Relationship Intent 
Faith Journey 

Example structure: 

Values 
• Family-oriented 
• Growth-minded 
• Loyal partnership 

Relationship Intent 
• Long-term commitment 
• Marriage-focused 

Faith Journey 
• Faith is an important part of life 

Microcopy below: 

Here, people meet you through meaning — not just images. 

 

Photo Reveal (Step 2) 

After a short fade animation (about 1–2 seconds), the profile photo should gently appear at the top of the card. 

This reinforces the platform’s philosophy: 

Values and direction are considered before physical appearance. 

 

Interaction Buttons 

After the reveal, the user sees: 

💛 Say Yes 
🌿 Explore Slowly 
✨ Not Aligned 

 

UX Philosophy 

This values-first reveal is intentional and differentiates SayYes Connect from swipe-based dating apps. 

It encourages users to consider: 

compatibility 

shared values 

long-term direction 

before focusing on appearance.
